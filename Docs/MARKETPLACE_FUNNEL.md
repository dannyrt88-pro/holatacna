# Marketplace Funnel

## Funnel general del marketplace

```txt
Turista chileno
  ->
Contenido redes sociales / SEO
  ->
Landing pages
  ->
Lead capturado
  ->
Routing de proveedores
  ->
Proveedor recibe lead
  ->
Servicio realizado
```

## 1. Turista chileno

El usuario objetivo es una persona de Chile interesada en:

- tratamientos medicos
- servicios aliados del viaje
- opciones mas claras o convenientes en Tacna

## 2. Contenido de descubrimiento

La demanda se activa antes del formulario.

Los principales disparadores del funnel son:

- redes sociales
- contenido SEO
- comparativas de precio o ciudad
- contenidos educativos por tratamiento

## 3. Landing pages

Las landings convierten interes en intencion concreta.

Su funcion es:

- explicar el servicio
- construir confianza
- reducir friccion
- capturar el lead

## 4. Lead capturado

Cuando el usuario completa el formulario:

- el lead entra a `create-lead`
- se normaliza el payload
- se calculan datos comerciales basicos
- se dispara el motor de routing

## 5. Routing de proveedores

Aqui el sistema conecta oferta y demanda.

El routing:

- filtra providers compatibles
- ordena candidatos
- autoasigna o deja pendiente
- guarda sugerencia y trazabilidad

## 6. Proveedor recibe lead

El provider recibe un lead mas estructurado:

- con servicio identificado
- con contexto de ciudad
- con sugerencia o asignacion previa

Eso reduce desorden operativo y acelera la derivacion comercial.

## 7. Servicio realizado

El funnel ideal termina cuando:

- el provider atiende al lead
- el usuario concreta el servicio
- el sistema puede medir calidad de asignacion y cobertura

## Como conecta oferta y demanda

HolaTacna actua como capa de orquestacion entre:

- demanda captada desde contenido y SEO
- oferta de providers configurados operativamente
- equipo comercial que supervisa y corrige asignaciones

## Valor del funnel

El funnel no solo sirve para captar leads.
Tambien sirve para:

- distribuir mejor la demanda
- entender que providers responden mejor
- detectar servicios con poca cobertura
- alimentar mejoras futuras del ranking
